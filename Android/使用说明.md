# 禄命法 APK 使用说明

## 一、应用简介

禄命法是一款基于中国传统命理学的八字排盘系统，可完全离线运行，无需网络连接。

## 二、主要功能

### 2.1 八字排盘
- 支持公历和农历两种日期输入方式
- 自动计算天干地支、十神、纳音、神煞等
- 支持经纬度设置（真太阳时计算）

### 2.2 记录管理
- 自动保存排盘记录
- 支持按日期类型筛选（公历/农历/四柱）
- 支持按姓名搜索
- **滑动操作**：向左滑动记录卡片可查看「笔记」和「删除」按钮
- 笔记功能：可对每条记录添加个人笔记

### 2.3 大运小运
- 自动计算大运、小运、流年
- 支持查看任意年龄段的大运流年详情
- 神煞排列分析

## 三、使用方法

### 3.1 新建排盘
1. 点击底部导航「排盘」
2. 选择性别（男/女）
3. 选择日期类型（公历/农历）
4. 选择出生日期和时间
5. 点击「开始排盘」

### 3.2 查看详情
1. 进入「记录」页面
2. 点击任意记录卡片进入专业排盘页面

### 3.3 记录笔记
1. 在记录页面，向左滑动记录卡片
2. 点击「笔记」按钮
3. 输入笔记内容
4. 点击「保存」

### 3.4 删除记录
1. 向左滑动记录卡片
2. 点击「删除」按钮
3. 在确认对话框中点击「删除」

## 四、设置说明

### 4.1 时区设置
- 默认使用北京时间（UTC+8）
- 可选择「自动」跟随手机系统时区

### 4.2 经度设置
- 用于计算真太阳时
- 默认使用当地经度
- 可手动输入或自动定位

## 五、注意事项

1. 本应用完全离线运行，无需网络连接
2. 所有数据存储在本地设备
3. 卸载应用将清除所有记录和笔记
4. 请勿在未经授权的情况下传播应用
