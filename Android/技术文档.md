# 禄命法 APK 技术文档

## 一、项目架构

### 1.1 整体架构

```
APK → NanoHTTPD 本地服务器 → WebView 加载前端 → JS 算法本地计算
```

### 1.2 技术栈

| 组件 | 技术方案 |
|------|---------|
| 后端 | NanoHTTPD 2.3.1 (静态文件服务) |
| 算法 | lunar-javascript + 自研JS算法 |
| 前端 | HTML/CSS/JavaScript |
| 存储 | IndexedDB (本地数据库) |
| 通信 | 前端直接调用本地JS算法 |

## 三、核心模块

### 3.1 MainActivity

**职责**：
- 启动 LocalServer（端口8080）
- 配置 WebView
- 加载本地前端页面

**关键代码**：
```java
localServer = new LocalServer(8080, getAssets());
localServer.start();

webView.getSettings().setJavaScriptEnabled(true);
webView.getSettings().setDomStorageEnabled(true);
webView.loadUrl("http://localhost:8080/首页排盘/index.html");
```

### 3.2 LocalServer

**职责**：
- 继承 NanoHTTPD
- 从 assets/web/ 目录提供静态文件
- 处理 MIME 类型

**支持的 MIME 类型**：
- .html → text/html
- .js → application/javascript
- .css → text/css
- .png → image/png
- .jpg → image/jpeg
- .json → application/json
- .woff/.woff2/.ttf → 字体文件

### 3.3 算法模块

| 文件 | 职责 |
|------|------|
| lunar-wrapper.js | 封装 lunar-javascript 库 |
| bazi.js | 八字排盘核心算法 |
| liunian.js | 流年流月流日计算 |
| dayun.js | 大运小运计算 |
| shensha.js | 神煞排列计算 |
| xingyao.js | 星曜计算 |
| nayin.js | 纳音五行十二长生 |

### 3.4 数据库

**使用 IndexedDB**，数据库名：`luming_db`

**记录结构**：
```javascript
{
  id: number,           // 自动递增ID
  name: string,         // 姓名
  gender: string,       // 性别（男/女）
  dateType: string,    // 日期类型（solar/lunar）
  solarDate: string,    // 阳历日期
  lunarDate: string,    // 农历日期
  year: string,         // 年柱干支
  month: string,        // 月柱干支
  day: string,          // 日柱干支
  hour: string,         // 时柱干支
  zodiac: string,       // 生肖
  fullData: object,     // 完整排盘数据
  note: string,         // 用户笔记
  createdAt: string,    // 创建时间
  updatedAt: string     // 更新时间
}
```

| 权限 | 用途 |

## 五、构建说明

### 5.1 环境要求
- Android Studio Arctic Fox 或更高版本
- JDK 1.8+
- Android SDK 24+ (Android 7.0)

### 5.2 构建命令

```bash
cd Android
./gradlew assembleDebug    # Debug 构建
./gradlew assembleRelease   # Release 构建
```

### 5.3 输出位置
- Debug APK: `app/build/outputs/apk/debug/app-debug.apk`
- Release APK: `app/build/outputs/apk/release/app-release.apk`

## 六、已知限制

1. 不支持 WebSQL（已使用 IndexedDB 替代）
2. 部分浏览器可能不支持 ES6 模块动态导入
3. 需要启用 JavaScript 才能正常运行
