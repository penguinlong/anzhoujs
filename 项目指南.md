# 禄命法 - 项目总览

## 一、项目简介

禄命法是一款基于中国传统命理学的八字排盘系统，采用 Android Native + WebView 混合架构，可完全离线运行。

## 二、技术架构

```
APK → NanoHTTPD 本地服务器 → WebView 加载前端 → JavaScript 算法本地计算
```

| 组件 | 技术方案 |
|------|---------|
| 后端 | NanoHTTPD 2.3.1 (静态文件服务) |
| 算法 | lunar-javascript + 自研 JS 算法 |
| 前端 | HTML/CSS/JavaScript |
| 存储 | IndexedDB (本地数据库) |
| 构建 | Gradle 8.2 |

## 四、核心功能

### 4.1 八字排盘
- 支持公历和农历两种日期输入
- 自动计算天干地支、十神、纳音
- 支持经纬度设置（真太阳时计算）

### 4.2 神煞系统
- 驿马、华盖、桃花、劫煞、灾煞
- 飞刃、截路空亡、禄马同乡、背禄逐马
- 天德合、月德合、太极贵、文星贵等

### 4.3 大运小运流年
- 基于《五行精纪》古法计算
- 支持童限、大运（12个）、小运（120岁）
- 流年流月流日详细分析

### 4.4 记录管理
- IndexedDB 本地存储
- 支持按日期类型筛选
- 笔记功能

## 五、算法模块

| 文件 | 职责 |
|------|------|
| `lunar-wrapper.js` | 封装 lunar-javascript 库 |
| `bazi.js` | 八字排盘核心算法 |
| `liunian.js` | 流年流月流日计算 |
| `dayun.js` | 大运小运计算 |
| `shensha.js` | 神煞排列计算 |
| `nayin.js` | 纳音五行、十二长生 |
| `xingyao.js` | 星曜计算 |

## 七、快速开始

### Web 开发调试
```bash
cd Android/app/src/main/assets/web
python -m http.server 8080
# 访问 http://127.0.0.1:8080/首页排盘/index.html
```

### Android 构建
```bash
cd Android
gradle assembleDebug
```
