UI 交互与功能优化需求文档
一、文档概述
本文档为干支排盘系统 UI 交互样式优化、功能逻辑修复及校验规则完善需求说明，面向开发人员，明确界面改造、历法修复、干支联动规则、排盘功能等核心要求，确保界面规范、逻辑准确、交互流畅。
二、UI 交互与样式优化（核心设计）
2.1 日期 / 时间选择器改造
彻底移除原生日期 / 时间下拉选择器
禁止使用系统原生下拉框，避免与干支滚轮选择器（wheel picker）发生样式冲突、交互冲突。
干支模式隐藏时间选择器
四柱 / 干支输入模式下，默认不显示时、分时间选择器；
可保留「显示时柱」开关，默认状态为关闭。
统一沿用干支选择器样式规范
采用固定布局：4 列 × 2 行圆形按钮网格
4 列分别对应：年柱、月柱、日柱、时柱
每列包含 2 个圆形按钮：天干 + 地支
点击按钮弹出干支选择浮层
天干浮层：2 行 × 5 列
地支浮层：3 行 × 4 列
要求无布局错位、无多余冗余元素。
2.2 干支选择器布局规范
标准布局还原
年柱、月柱、日柱、时柱 4 个标签横向等间距排列
下方 8 个圆形按钮严格按「每柱 2 个」整齐排布
所有按钮尺寸统一、水平垂直对齐一致
冗余元素清理
删除列外独立按钮、多余分割线与占位控件
修复按钮错位、重叠、偏移问题
保证点击区域完整、交互逻辑正常。
三、功能逻辑与校验规则（核心业务）
3.1 农历日期选择器修复
补充闰月选项
1978 年（戊午年）农历存在闰八月，需在月份列表中新增独立「闰 8 月」选项
确保用户可正常选择闰月对应日期。
农历历法准确性
按真实农历历法规则完整适配所有历史年份闰月，不得出现漏项、错项、重复项。
3.2 干支联动校验规则（核心逻辑）
年柱 → 月柱联动（五虎遁元）
月干由年干自动推导，仅展示合法月干，非法组合置灰不可选
若用户先选月柱后选年柱，出现非法组合时：
自动清空非法项
给出清晰文字提示
日柱 → 时柱联动（五鼠遁元）
时干由日干自动推导，仅展示合法时干，非法组合置灰不可选
若用户先选时柱后选日柱，出现非法组合时：
自动清空非法项
给出清晰文字提示
60 甲子合法性校验
年柱、日柱仅允许 60 甲子范围内有效组合
非法组合直接禁用，不可选中。
3.3 排盘功能修复
修复滚轮选择器触发排盘无响应、不跳转问题
确保新时间选择器所有参数正确传递：
公历日期
农历日期（含闰月）
天干地支组合
点击「排盘」按钮后：
正常跳转排盘结果页
正确渲染四柱、大运、流年等排盘内容
无控制台报错、无页面卡顿、无数据异常。
四、补充细节要求
4.1 交互反馈优化
干支选择浮层中，非法组合直接置灰不可点击
存在未修复的非法干支组合时：
浮层「确定」按钮置灰禁用
同步显示提示语，告知用户组合不合法
4.2 样式一致性
干支圆形按钮、选择浮层、柱位标签样式1:1 还原设计图
禁止出现样式错乱、颜色偏差、字体不一致、圆角 / 阴影不统一等问题
4.3 功能完整性
干支模式、农历模式、滚轮选择器模式均需完整可用
三种模式下排盘、重置、切换、保存等相关功能正常运行
跨模式切换不丢失数据、不出现异常报错。
五、验收标准
UI 布局无错位、无重叠、无冗余元素，样式与设计一致
原生选择器已移除，干支模式时间选择器默认隐藏
农历闰月完整适配，1978 闰八月可正常选择
五虎遁元、五鼠遁元联动规则生效，非法干支自动置灰 / 清空提示
年柱、日柱严格遵循 60 甲子规则，非法组合不可选
排盘按钮点击响应正常，参数传递正确，排盘结果渲染无误
所有模式功能完整，交互反馈清晰，无报错、无卡顿。
